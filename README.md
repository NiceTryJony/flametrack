# FlameTrack 🔥

Real coding streaks — earned by successful builds, tests, and deploys.
Not by staring at your editor.

## Features
- Tracks successful builds, tests, and git pushes
- Heatmap of activity over 12 months
- XP, levels, and achievements
- Streak protection on weekends

## Usage
Open the stats panel: `Ctrl+Alt+F` (or `Cmd+Alt+F` on Mac)
